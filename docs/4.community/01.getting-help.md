# Getting Help

If you are blocked, start with these steps:

1. read the Getting Started and Directory Structure docs
2. compare your app with `repos/vuloom/examples/zero-config`
3. confirm `vuloom prepare` has been run
4. reduce the issue to the smallest reproducible route or middleware case

For project discussion and support:

- GitHub: `hornjs/vuloom`
- Discord: `https://discord.hornjs.dev`
