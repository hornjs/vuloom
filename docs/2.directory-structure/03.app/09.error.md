# `app/error.vue`

This is the app-level error fallback.

Use it for failures that are not handled by a nearer route-level `error.*` boundary.

Typical responsibilities:

- display a generic recovery screen
- show request or route metadata in development
- provide a safe fallback for unexpected render errors
