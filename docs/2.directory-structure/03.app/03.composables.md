# `app/composables`

Use this folder for app-local `use*` helpers that compose your project logic.

Examples:

- `useTheme()`
- `useCurrentUser()`
- `useFeatureFlags()`

Framework composables such as `useAppData()`, `useLoaderData()`, and `useNavigation()` still come from `vuloom/app`.
