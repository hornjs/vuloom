# `app/app.vue`

`app/app.vue` is the document shell.

It must render:

- `<html>`
- `<head>`
- `<body>`

Common responsibilities:

- global CSS imports
- root providers
- shared shell markup
- reading `useAppData()`
- applying `useHead()` defaults

This file is server-rendered and hydrated as the top-level document.
