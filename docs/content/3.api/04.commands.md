# Commands

Phial ships a small CLI with four core commands.

## `phial dev`

Start the development server.

```bash
phial dev
phial dev . --port 3000 --host 0.0.0.0
```

## `phial prepare`

Generate `.phial/types/`.

```bash
phial prepare
```

Use this whenever route files or middleware names change.

## `phial build`

Build the production client and server output.

```bash
phial build
phial build . --mode production
phial build . --watch
```

## `phial start`

Serve the result of `phial build`.

```bash
phial start
phial start . --port 3000 --host 0.0.0.0
```

## Shared flags

- root path as the first positional argument
- `--config`
- `--mode`

`dev` and `start` also support `--host` and `--port`.
