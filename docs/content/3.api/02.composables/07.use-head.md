# `useHead()`

`useHead()` is available from `phial/app`.

It manages document head state through the underlying `vuepagelet` runtime and supports both browser updates and SSR output.

## Related helpers

- `useTitle()`
- `useMeta()`
- `useLink()`
- `useStyle()`
- `useScript()`
- `updateHead()`

## Intended scope

- `<title>`
- `<meta>`
- `<link>`
- `<style>`
- `<script>`
- `<html>` attributes
- `<body>` attributes
