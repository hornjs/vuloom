# `useState()` and `useCookie()`

## `useState()`

`useState()` creates framework-managed shared state.

```ts
const count = useState('count', () => 0)
```

### Behavior

- server-side state is request-scoped
- client state is hydrated from the SSR payload
- the same key resolves the same shared ref within the app

## `useCookie()`

`useCookie()` reads and writes a browser cookie through a Vue ref.

```ts
const theme = useCookie('theme')
```

### Options

- `path`
- `expires`
- `domain`
- `secure`
- `sameSite`
- `maxAge`

Use `useCookie()` for client cookie state. Use server loaders or middleware when the value must be enforced or read on the server.
