# `.gitignore`

A Phial app should usually ignore:

- `node_modules/`
- `.phial/`
- `.output/`
- `*.tsbuildinfo`
- local env overrides such as `.env.local`

The exact list depends on your toolchain, but `.output/` and `.phial/` should almost always be treated as generated output. If you explicitly configure the client build to use `dist/`, ignore that directory as well.
