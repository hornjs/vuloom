# `.phial`

`.phial/` is generated by Phial. Treat it as build output, not source code.

## What goes here

- `.phial/types/*.d.ts` from `phial prepare`

## What it is used for

- route and middleware type generation
- internal framework bootstrap files

## Should it be committed?

Usually no. Add `.phial/` to `.gitignore`.

## When it changes

- `phial prepare`
