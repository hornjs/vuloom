# `.output`

`.output/` is the default production build output.

Phial writes:

- client assets to `.output/public/`
- the production server bundle to `.output/server/index.js`
- `.output/public/.vite/manifest.json` for client asset resolution

Phial does not use `.output/` as source code. Delete and rebuild it at any time.

Use `phial start` after `phial build` to serve the built client output together with the server bundle in `.output/server/`.
