# `tsconfig.json`

Your app `tsconfig.json` should include Phial's generated types.

```json
{
  "include": [
    "app",
    "server",
    "phial.config.ts",
    ".phial/types/**/*.d.ts"
  ]
}
```

Why this matters:

- route ids and route params come from `.phial/types/routes.d.ts`
- app and server middleware names come from `.phial/types/middleware.d.ts`

Run `phial prepare` whenever the route or middleware graph changes so these files stay current.
