# middleware

`server/middleware` contains named reusable server middleware implementations.

These names can be referenced from:

- `phial.config.ts -> server.middleware`
- `server/routes/**/_middleware.ts`
- `server/routes/**/*.ts -> export default { middleware: [...] }`

Execution order is fixed:

1. global server middleware from `phial.config.ts`
2. outer-to-inner middleware from `server/routes/**/_middleware.ts`
3. route-local middleware on the selected server route module
4. the selected route handler
