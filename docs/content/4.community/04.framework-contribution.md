# Framework Contribution

Phial is organized as a workspace of focused packages.

Important packages include:

- `phial`
- `create-phial`

When changing framework behavior:

- update the relevant workspace package
- keep `repos/phial` aligned with its generated runtime layers and Vite integration
- update `examples/zero-config`
- update `repos/create-phial` templates
- run the smoke scripts that cover the affected area
